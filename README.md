# Hash Node Tree Visualizer
A addon I made for Godot 4.6 to automatically create a visual replica of the scene tree using Control nodes.
## Why?
I use [excalidraw](https://excalidraw.com/) to create diagrams all the time. I found myself constantly creating mock node trees to help visually explain how systems should be developed to my programming team. I was tired of making new node trees every time and figured I could just make a tool that does it for me

## How To Use?
Add a NodeTreeVisualizer node to the scene tree. By default it will build the visual tree off of it's parent, but the node it starts from can be changed
